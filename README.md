# مسار | Masar 🚀

تطبيق "مسار" هو مدير مشاريع شخصي بسيط وقوي للمطورين، يهدف إلى تنظيم المشاريع البرمجية وتتبع الإصدارات وتدوين الملاحظات البرمجية في مكان واحد.

هذا المشروع هو تطبيق ويب عصري (React PWA) تم تنظيفه من بقايا ملفات Flutter ليركز بالكامل على تقنيات الويب الحديثة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

### الواجهة الأمامية (Frontend):
- **React 18** مع **TypeScript**
- **Vite** (كأداة بناء سريعة)
- **Tailwind CSS** للتصميم الجمالي والمتجاوب
- **Lucide React** للأيقونات
- **React Router DOM** للتنقل بين الصفحات

### الخلفية (Backend):
- **Node.js** و **Express.js**
- **SQLite3** لقاعدة البيانات المحلية
- **UUID** لإنشاء معرفات فريدة

---

## 🚀 طريقة التشغيل (How to Run)

### 1. تشغيل الـ Backend
الـ Backend مسؤول عن حفظ البيانات في قاعدة بيانات SQLite المحلية.

```bash
# تأكد من تثبيت الحزم أولاً
npm install

# تشغيل السيرفر
npm run server
```
سيعمل السيرفر على الرابط `http://localhost:3001`.

### 2. تشغيل الـ Frontend
الواجهة الأمامية توفر تجربة المستخدم التفاعلية.

```bash
# في نافذة terminal منفصلة
npm run dev
```
سيعمل التطبيق على الرابط `http://localhost:3000`.

---

## 📁 هيكل المجلدات (Folder Structure)

- `backend/`: يحتوي على كود السيرفر وقاعدة البيانات `schema.sql`.
- `src/services/`: يحتوي على `api.ts` لربط الواجهة بالسيرفر.
- `pages/`: جميع صفحات التطبيق (الرئيسية، المشاريع، الإحصائيات، إلخ).
- `components/`: المكونات المشتركة مثل `Layout`.
- `types.ts`: تعريفات TypeScript للمشروع.

## 📱 ميزات الـ PWA
التطبيق جاهز ليعمل كتطبيق ويب تقدمي (PWA)، مما يسمح بتثبيته على سطح المكتب أو الهاتف والوصول السريع إليه.
